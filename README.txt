Welcome to the Termy Package Repository!

Termy is a package manager for Windows, based in the command line. It is simple, with only 3 commands to use.

All packages in the repository are checked well for anything that could cause damage to your system. Nobody besides me can post new
packages. To fill out a form to get your package checked, and maybe uploaded, fill out a Google Form. The link is below:
https://forms.gle/VU5ZxiT7fze1CP1E7 

All packages in the repository are allowed to be copied and modified!
